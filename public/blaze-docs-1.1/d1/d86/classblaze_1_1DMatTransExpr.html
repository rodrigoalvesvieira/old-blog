Detailed Description

template<typename MT, bool SO>
class blaze::DMatTransExpr< MT, SO >

Expression object for dense matrix transpositions.

The DMatTransExpr class represents the compile time expression for transpositions of dense matrices.

Member Enumeration Documentation

Compilation switch for the evaluation strategy of the transposition expression.

The useAssign compile time constant expression represents a compilation switch for the evaluation strategy of the transposition expression. In case the given dense matrix expression of type MT requires an intermediate evaluation, useAssign will be set to 1 and the transposition expression will be evaluated via the assign function family. Otherwise useAssign will be set to 0 and the expression will be evaluated via the subscript operator.
