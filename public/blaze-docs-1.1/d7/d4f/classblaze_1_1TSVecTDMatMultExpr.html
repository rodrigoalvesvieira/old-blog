Detailed Description

template<typename VT, typename MT>
class blaze::TSVecTDMatMultExpr< VT, MT >

Expression object for transpose sparse vector-transpose dense matrix multiplications.

The TSVecTDMatMultExpr class represents the compile time expression for multiplications between transpose sparse vectors and column-major dense matrices.

Compilation switch for the evaluation strategy of the multiplication expression.

The useAssign compile time constant expression represents a compilation switch for the evaluation strategy of the multiplication expression. In case the dense matrix expression requires an intermediate evaluation or the sparse vector expression is a compound expression, useAssign will be set to true and the addition expression will be evaluated via the assign function family. Otherwise useAssign will be set to false and the expression will be evaluated via the subscript operator.
