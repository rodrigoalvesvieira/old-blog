About

The author

Rodrigo Vieira is a 17 years old developer, writing mostly Ruby and JavaScript code, interested in all things programming, specially Web application development and the Unix operating system.

Rodrigo loves open source software and, when he can, he contributes a little bit to a project. When this happens his GitHub records it.

You can follow him on Twitter and, of course, subscribe to his blog's feed.

The blog

This blog is intended to publish mostly technical content on the author´s current interests. However, the author may write in a different style occasionally.

Visit the Archive to browse through all the blog posts.

All posts will be written in English. However, English is not the author´s primary language, so be friendly and send him a comment if you see a typo or any kind of grammatical error.

This blog was made with Jekyll and is hosted on GitHub Pages. The blog was designed by Rodrigo and is probably the best thing he will ever do with CSS. Judge from that perspective.
